# System Specification

## Hardware Requirements
- **Minimum**: 4-Core CPU, 8GB RAM.
- **Recommended**: 8-Core CPU, 16GB RAM, NVIDIA T4 or equivalent GPU.

## Software Dependencies
- OS: Linux (Ubuntu 22.04 recommended) or Windows 11.
- Python 3.11
- NVIDIA Driver > 525, CUDA 11.8+, cuDNN 8.x (for GPU mode).
