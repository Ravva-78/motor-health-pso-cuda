# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-23
### Added
- Complete end-to-end telemetry ingestion pipeline via MQTT.
- ZeroMQ IPC messaging architecture for decoupled inference.
- Autonomous Particle Swarm Optimization (PSO) using numba.cuda.
- Live Streamlit dashboard for real-time monitoring and explanations.
- Graceful CPU fallback mechanism for missing NVIDIA binaries.
- Real-time SHAP feature contribution analysis.
