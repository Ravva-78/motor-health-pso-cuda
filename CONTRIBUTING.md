# Contributing to AeroForge

First off, thank you for considering contributing to AeroForge! 

## Development Process
1. Fork the repo and create your branch from main.
2. Ensure you have the equirements.txt installed.
3. Write clear, readable code and include tests where applicable.
4. Issue that pull request!

## Code Style
Please adhere to PEP-8 guidelines. Avoid introducing breaking architectural changes without prior discussion in an Issue.
